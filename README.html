RaboGRAPH

RaboGRAHP is een programma wat statistieken, overzichten en grafieken genereerd van je Rabobank afschriften. Deze afschriften kun je downloaden via Rabobank Internetbankieren.

Auteur: Ferry Boemder
Versie: 0.5
Licensie: GNU GPL (General Public License)
Compatibility: Unix, Windows, (alles wat php kan draaien)
Homepage: http://www.electricmonk.nl

Wat doet het?

RaboGRAP genereerd ruwweg de volgende dingen:

  • Voor alle maanden, jaren en het totaal aantal transacties:
    • Het totaal aantal transacties (bij- en afschrijvingen)
    • De som en verschillen van bedragen van alle transacties (alle, bij- en afschrijvingen)
    • De som en verschillen van bedragen van alle transacties (alle, bij- en afschrijvingen) per type (bijv. 'Acceptgiro' of 'Machtiging')
    • Een overzicht van transacties (incl. aantal en bedrag som) per tegenpartij (bijv. 'Agis zorgverzekeringen')
    • Een overzicht van transacties (incl. aantal en bedrag som/verschil) per datum (incl. totalen p/m, etc)
  • Voor alle maanden van elk jaar:
    • Een grafiek met een overzicht van de bij- en afschrijvingen van elke maand.

Benodigheden

Unix

  • PHP Commandline interpreter
  • Webbrowser
  • Rabobank Internet bankieren

Windows

  • Webbrowser
  • Rabobank Internet bankieren

Installatie / gebruik

Unix

  • Untar het rabograp-X.Y.tar.gz archief
  • Start Rabobank Internet bankieren en log in.
  • Ga naar Betalen en Sparen → Download
  • Maak bij Alle transacties van de volgende rekening: een keuze over welke rekening je statiesteken wilt geneneren.
  • Voer een begin en einddatum in. Als begindatum kun je 01-01-2004 invullen (eerder kan niet, bij mij althans). Als einddatum vul je de datum van vandaag in.
  • Kies Kommagescheiden formaat en klik op de download knop.
  • Sla het resulterende bestand 'mut.txt' op in de rabograp-X.Y/ directory.
  • Run het rabograp.php script: php ./rabograp.txt
  • Open met je browser het rabograp.html bestand om de index pagina te laden.

Windows

  • Download en run het rabograp-X.Y.exe bestand. Het bestand maakt nu een directory aan met daarin het programma.
  • Start Rabobank Internet bankieren en log in.
  • Ga naar Betalen en Sparen → Download
  • Maak bij Alle transacties van de volgende rekening: een keuze over welke rekening je statiesteken wilt geneneren.
  • Voer een begin en einddatum in. Als begindatum kun je 01-01-2004 invullen (eerder kan niet, bij mij althans). Als einddatum vul je de datum van vandaag in.
  • Kies Kommagescheiden formaat en klik op de download knop.
  • Sla het resulterende bestand 'mut.txt' op in de rabograp-X.Y/ directory.
  • Dubbelklik het 'rabograp.bat bestand.
  • Heb even geduld (het programma is wat traag onder Windows). Nu start automatisch een INternet browser op met daarin de hoofdpagina van de rapportage.
